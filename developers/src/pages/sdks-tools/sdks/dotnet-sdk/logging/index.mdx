# {frontmatter.title}

The SDK uses `Microsoft.Extensions.Logging` to perform all of its logging
tasks. To configure logging for your app simply create a new ILoggerFactory
and call the LogProvider.SetLogFactory() method to tell the Deepgram library
how to log.

For example, to log to the console with serilog you can do the following:

```csharp
using Microsoft.Extensions.Logging;
using Microsoft.Extensions.Configuration;
using Deepgram.Logger;
using Serilog;

var log = new LoggerConfiguration()
    .MinimumLevel.Debug()
    .WriteTo.Console(outputTemplate: "{Timestamp:HH:mm} [{Level}]: {Message}\n")
    .CreateLogger();
var factory = new LoggerFactory();
factory.AddSerilog(log);
LogProvider.SetLogFactory(factory);
```
