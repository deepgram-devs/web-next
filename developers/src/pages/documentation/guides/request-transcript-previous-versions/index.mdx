# {frontmatter.title}

Deepgram releases language and use case models at various times throughout the year. If you want to revert to an earlier model, you can pass the `version` parameter in your request to the Deepgram API.

## Recently Updated Language Versions

Deepgram has recently released the following versions.

<Alert type="info">

Not all languages have recent updates. To learn about all the languages that Deepgram offers, see [Features: Language](/documentation/features/language/).

</Alert>
