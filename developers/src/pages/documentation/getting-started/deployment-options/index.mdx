# {frontmatter.title}

Deepgram offers the following deployment options:

| **Option**   | **Description**                                                                                                                                                                                                                                                                                 |
| ------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Public Cloud | A multi-tenant cloud service running on Deepgram's cloud with standard authentication and customization features.                                                                                                                                                                               |
| On Premises  | A dedicated service deployed to customer-requisitioned cloud instances, such as Amazon Web Services (AWS) or Google Cloud Platform (GCP), or customer data centers. The On-Premises option is available for customers who cannot use a multi-tenant public cloud service in their organization. |
