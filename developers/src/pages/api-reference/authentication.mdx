## Endpoint

`https://api.deepgram.com/v1`

## Authentication

<div class="grid gap-6 lg:grid-cols-2">Send requests to the API with an `Authorization` header that references your project's API Key:</div>
 
<div class="grid lg:grid-cols-2 gap-6">
`Authorization: Token <YOUR_DEEPGRAM_API_KEY>`
</div>

<div class="grid gap-6 lg:grid-cols-2">
	You can [create a Deepgram API Key in the Deepgram Console](/documentation/getting-started/authentication/). You must create your first API Key using the Console.
</div>

<div class="grid gap-6 lg:grid-cols-2">
	All API requests must be made over HTTPS. Calls made over plain HTTP will fail. API requests made without authentication will also fail.
</div>
